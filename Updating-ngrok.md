`ngrok` ia a _utility software_ it can be downloaded and used directly from the location of the executable **without** requiring previous installation

as at 2024-02-21, the ngrok version shipped with `Laragon v6.0.0` (latest) is out of date, in order to avoid errors, we need to update it

go to [ngrok > Downloads](https://ngrok.com/download) to download the latest Agent

we want the Windows x64 zip file - it only contains the executable utility

Once you have the zip, extract it to `{Laragon Root}/bin/ngrok` replacing the old file
